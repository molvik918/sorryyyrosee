# I AM SORRY ROSE BABE 💕

A heartfelt apology website created with love and sincerity.

## Features

✨ **Beautiful Design**
- Gradient purple background
- Smooth animations and transitions
- Responsive design for all devices
- Falling hearts animation

🎵 **Background Music**
- Soft, emotional background music theme
- Play/Pause controls
- Auto-plays on page load (if browser allows)

💌 **Personal Message**
- Heartfelt apology message
- Animated text reveals
- Floating hearts animation

## How to Use

1. Open `index.html` in your web browser
2. Click the "🎵 Play Music" button to play/pause background music
3. Enjoy the animations and message

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and animations
- `script.js` - Interactive elements and music control

## Browser Compatibility

Works best on:
- Chrome/Chromium
- Firefox
- Safari
- Edge

Background music is sourced from royalty-free music services.
